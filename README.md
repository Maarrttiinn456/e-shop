# E-shop

Next.js e-shop postavený na App Routeru s TypeScriptem a Tailwind CSS.

## Spuštění

```bash
npm run dev
```

Otevři [http://localhost:3000](http://localhost:3000) v prohlížeči.

## Příkazy

```bash
npm run dev      # vývojový server
npm run build    # produkční build
npm run lint     # ESLint
npm run test     # testy
```

## Integrace

- **Produkty:** načítány z API [dummyjson.com](https://dummyjson.com)
- **Platební brána:** GoGout (testovací prostředí)

## Produkční nasazení

E-shop běží na vlastním webserveru na adrese [https://eshop.m-sch.cz/](https://eshop.m-sch.cz/).
